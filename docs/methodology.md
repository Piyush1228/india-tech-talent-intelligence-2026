Tech Role Classification Methodology
The source dataset (Naukri.com, 97,105 cleaned records) spans all industries, not just technology. A keyword-based classifier was built using an include/exclude system with word-boundary regex matching to prevent substring false-positives (e.g., "Sales" incorrectly matching "Salesforce"). The classifier was iteratively refined across 3 versions, each validated against spot-checks of top-N classified titles and a "false negative" audit (tech-sounding titles that were excluded). Final classification yielded 32,438 tech-specific postings (33.4% of cleaned data). The method prioritizes precision over recall — ambiguous generic titles (e.g., bare "Analyst," "Salesforce Administrator") were deliberately excluded to avoid contaminating the dataset with adjacent BPO, finance, or functional-consultant roles. This represents an estimated 90%+ precision classification.
